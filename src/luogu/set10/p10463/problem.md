# P10463

给定一个长度为 $N$ 的数列 $a$，以及 $M$ 条指令。每条指令为以下两种之一：

- **C l r d**：将 $a_l, a_{l+1}, \ldots, a_r$ 都加上 $d$。
- **Q l r**：询问 $a_l, a_{l+1}, \ldots, a_r$ 的最大公约数（$\gcd$）。

对于每个 **Q** 询问，输出一个整数表示答案。

## 输入格式

第一行两个整数 $N, M$。

第二行 $N$ 个整数，依次为 $a_1, a_2, \ldots, a_N$。

接下来 $M$ 行，每行一条指令，格式见题目描述。

## 输出格式

对于每个询问，输出一行一个整数表示答案。

## 样例

### 输入

```
5 5
1 3 5 7 9
Q 1 5
C 1 5 1
Q 1 5
C 3 3 6
Q 2 4
```

### 输出

```
1
2
4
```

## 说明 / 提示

对于 $100\%$ 的数据：

- $N \le 5 \times 10^5$
- $M \le 10^5$
- $1 \le a_i \le 10^{18}$
- $|d| \le 10^{18}$

保证运算过程中数值在 `long long` 范围内。
