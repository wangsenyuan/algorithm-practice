# P1712 区间选择问题

## 题目描述

在数轴上有 n 个闭区间，从 1 至 n 编号，第 i 个闭区间为 [l<sub>i</sub>, r<sub>i</sub>]。

现在要从中选出 m 个区间，使得这 m 个区间共同包含至少一个位置。换句话说，就是使得存在一个 x，使得对于每一个被选中的区间 [l<sub>i</sub>, r<sub>i</sub>]，都有 l<sub>i</sub> ≤ x ≤ r<sub>i</sub>。

对于一个合法的选取方案，它的**花费**为被选中的最长区间长度减去被选中的最短区间长度。

区间 [l<sub>i</sub>, r<sub>i</sub>] 的长度定义为 (r<sub>i</sub> - l<sub>i</sub>)，即等于它的右端点的值减去左端点的值。

## 问题

求所有合法方案中最小的花费。如果不存在合法的方案，输出 -1。

## 分析

这是一个区间选择优化问题：

1. 需要选择 m 个区间使得它们有公共交集。
2. 目标是最小化选中区间中最长长度与最短长度的差值。
3. 如果无法找到 m 个有公共交集的区间，返回 -1。

## 思路 (Ideas)

1. 确定最大区间的情况下，看这个最大区间中，是否有一个点被其它 m-1 个区间覆盖。
