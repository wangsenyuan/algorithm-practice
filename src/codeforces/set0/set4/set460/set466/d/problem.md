# Problem

**题意：** 给定长度为 $n$ 的序列 $a_1, a_2, \ldots, a_n$ 和目标值 $h$。每次操作可选一段 $[l, r]$，将下标在 $[l, r]$ 内的元素各加 $1$。限制：任意两次操作的左端点两两不同，右端点也两两不同（即每个位置最多作为某次操作的左端点一次、作为某次操作的右端点一次）。求有多少种不同的操作方案，使得最终每个数都等于 $h$。两种方案不同当且仅当某次操作的区间在一种方案中出现而在另一种中未出现。答案对 $10^9 + 7$ 取模。

## Input

- 第一行：两个整数 $n$, $h$（$1 \le n, h \le 2000$）。
- 第二行：$n$ 个整数 $a_1, a_2, \ldots, a_n$（$0 \le a_i \le 2000$）。

## Output

一行一个整数，表示方案数对 $10^9 + 7$ 取模的结果。

## Examples

### Example 1

**Input:**

```
3 2
1 1 1
```

**Output:**

```
4
```

### Example 2

**Input:**

```
5 1
1 1 1 1 1
```

**Output:**

```
1
```

### Example 3

**Input:**

```
4 3
3 2 1 1
```

**Output:**

```
0
```

### ideas
1. 最终的高度应该是确定的
2. 每个位置只能被选中一次作为开始和结束，那么对于1来说，如果它被选中，那么它只能增加1
3. 所以，最后的结果要么是 h[0], 那么是 h[0] + 1 (同样的，可以对最后一个位置进行这样的处理)
4. 那么这样子，每个位置，需要增加的次数 d[i] = H - h[i] 
5. dp[i][j] 表示到i为止，有多少个还没有结束的区间, 如果 j + 1 = d[i], 那么这里需要开始
6. dp[i+1][j+1], 也可以马上结束 dp[i+1][j] 
7. 如果 j = d[i], 那么这里不能有开始， 但是可以结束 dp[i+1][j], dp[i+1][j-1]
8. 如果 j > d[i] => 0
