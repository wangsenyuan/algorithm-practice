# 题目描述

给你一个 m x n 的二维整数数组 `grid` 和一个整数 `k`。你从左上角的单元格 (0, 0) 出发，目标是到达右下角的单元格 (m - 1, n - 1)。

## 移动规则

有两种移动方式可用：

### 1. 普通移动
- 你可以从当前单元格 (i, j) 向右或向下移动
- 移动到 (i, j + 1)（右）或 (i + 1, j)（下）
- 成本为目标单元格的值

### 2. 传送
- 你可以从任意单元格 (i, j) 传送到任意满足 `grid[x][y] <= grid[i][j]` 的单元格 (x, y)
- 此移动的成本为 0
- 你最多可以传送 k 次

## 目标
返回从 (0, 0) 到达单元格 (m - 1, n - 1) 的**最小**总成本。


### example (k = 2)
[0,12,1,14,3],
[8,18,18,4,8],
[15,17,13,16,6],
[11,16,23,15,16],
[21,17,25,27,24],
[26,27,21,16,23],
[27,20,31,30,32]
