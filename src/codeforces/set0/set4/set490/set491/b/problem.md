# New York Hotel

## 题意

N×M 网格，相邻格点距离为 1（曼哈顿距离）。

给定 C 个点（酒店）和 H 个点（餐厅），选一个餐厅，使得所有酒店到该餐厅的**最大距离**最小。

输出：最小化的最大距离，以及达到该距离的餐厅编号（若有多个可输出任意一个）。

## Input

- 第一行：N, M (1 ≤ N, M ≤ 10^9)
- 下一行：C (1 ≤ C ≤ 10^5)，随后 C 行每行两个整数 x, y (1 ≤ x ≤ N, 1 ≤ y ≤ M)
- 下一行：H (1 ≤ H ≤ 10^5)，随后 H 行每行两个整数 x, y

多个点可能在同一格点。

## Output

第一行：最优距离。第二行：餐厅编号（从 1 开始）。

## Examples

### Example 1

**Input:**

```
10 10
2
1 1
3 3
2
1 10
4 4
```

**Output:**

```
6
2
```

## ideas
1. 因为没有block，所以，只需要考虑最外围的酒店就可以了
2. 这样的酒店最多只有4个
3. 然后检查所有酒店到它们的距离，找出最大距离种的最小值
