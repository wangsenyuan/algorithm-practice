# D - Repeated Sequence

Problem: [AtCoder Beginner Contest 384 D](https://atcoder.jp/contests/abc384/tasks/abc384_d)

## Problem Statement

An infinite sequence `A` has period `N`. Its first `N` terms
`A_1, A_2, ..., A_N` are given, and every later term repeats with the same
period.

Determine whether the sequence contains a non-empty contiguous subsequence
whose sum is exactly `S`.

## Constraints

- `1 <= N <= 2 * 10^5`
- `1 <= A_i <= 10^9`
- `1 <= S <= 10^18`
- All input values are integers.

## Input

```text
N S
A_1 A_2 ... A_N
```

## Output

Print `Yes` if a non-empty contiguous subsequence with sum `S` exists.
Otherwise, print `No`.

## Sample 1

```text
Input
3 42
3 8 4

Output
Yes
```

## Sample 2

```text
Input
3 1
3 8 4

Output
No
```

```text
Yes
```
